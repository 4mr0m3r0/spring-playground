## User:Password
Calling the endpoint with user and auto-generated password
```bash
curl -u user:<auto-generated-password> http://localhost:8080/hello
```

## Base64
```bash
# First encode user:password
echo -n user:<auto-generated-password> | base64
# Output e.g., dXNlcjo4NjJhNmUzYy0zMTE4LTQ4YWEtODQyNi1hZWEzMTQyMTNmOGQ=
# Then, add to the header
curl -H "Authorization: Basic dXNlcjo4NjJhNmUzYy0zMTE4LTQ4YWEtODQyNi1hZWEzMTQyMTNmOGQ=" http://localhost:8080/hello  
```
